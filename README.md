# react-clock

# Development

    # ライブラリのインストール(先にnpm install -g yarn)
    yarn install

    # ローカル起動
    yarn run start
    (localhost:8000で見ることができます)

    # ビルド
    yarn run build-dev (テスト用 ログとかでます)
    yarn run build-prd (本番用)
